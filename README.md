## 👋 About additive
Basic Minecraft modpack that includes many optimization and QoL mods.

## 📥 Download
You can install additive from Modrinth, Hangar, or download the packwiz file from [Releases](https://github.com/misilelab/additiveplus/releases) manually. This is the easiest way to install additive. We also offer a few other download methods if you don't want to use the default option.

### 🫥 Recommended
Download from **Modrinth** or **Hangar** (supports Minecraft 1.19.3 and later snapshots)
[![Modrinth](https://img.shields.io/badge/dynamic_json?color=006AFF&label=Additive&query=%24Additive%20&url=https://modrinth.com/modpack/additive)](https://modrinth.com/modpack/additive)
[![Hangar](https://img.shields.io/badge/dynamic_json?color=006AFF&label=Additive&query=%24Additive%20&url=https://hangar.mc](https://hangar.mc/modpacks/additive)](https://hangar.mc/modpacks/additive)

### 📁 Packwiz file
1. Download the packwiz file from [Releases](https://github.com/misilelab/additiveplus/releases)
2. Put it in your `.minecraft/packwiz` folder

## 🐛 Reporting issues
Please report issues at the [issue tracker](https://github.com/misilelab/additiveplus/issues) in the additive repository. Check if your issue is on the [list of known issues](https://github.com/intergrav/Adrenaline/wiki/Known-Issues) for Adrenaline first, as the issue might be caused by Adrenaline itself. If you're sure it's not an Adrenaline issue, please check the [adrenaline wiki](https://github.com/intergrav/Adrenaline/wiki) for troubleshooting. If you can't find a solution, open an issue in the additive repository. When reporting issues, please include:
- Minecraft version
- Additive version
- A list of all mods installed (even those not included with additive)
- A detailed description of the issue
- Logs (please put them in a code block or upload them to a [gist](https://gist.github.com/))
- Screenshots (if applicable)
